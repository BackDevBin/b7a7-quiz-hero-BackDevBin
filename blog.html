Difference Between : Map vs Foreach vs Filter vs Find


Map() :
The map() method returns a new array,The map() method is used to transform the elements of an array, The map() method can be used with other array methods
ForEach() :
forEach() method does not return a new array, forEach() method is used to loop through the elements of an array,forEach() method cannot be used with other array methods.
Filter():
filter() returns an array containing the element that satisfies the condition,In filter(), whole array is iterated despite the fact that the element being searched for is present at the beginning.
Find():
find() returns the element itself that satisfies the condition.find(), as soon as the element that satisfies the condition is found, it gets returned.
